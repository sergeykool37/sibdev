После запуска сервера django и создания миграции переходим по адресу "http://localhost:8000/",
выбираем  .csv файл для обработки, нажимаем кнопку "Confirm" для начала обработки.
После того как прошла обработка файла,переходим по адресу "http://localhost:8000/response", 
где отображается информация о 5ти клиентах, потративших наибольшую сумму за весь день.
